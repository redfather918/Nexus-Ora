> 版本：v3.4 | 更新日期：2026-06-13 | 状态：8 大模块全部开发完成

## 1. 产品概述

### 1.1 产品定位
Nexus Ora 是一款 **AI 原生玄学平台**，将东方传统命理学（八字排盘、五行分析）与现代 AI 技术结合，构建"灵境八部"产品矩阵：
- 灵境图谱 — 人生K线 / 五行平衡 / 八字排盘
- 灵境人格 — 十神虚拟伴侣 AI 对话
- 梦境回廊 — AI 梦境记录 / 可视化 / 解析
- 缘分配对 — 双人八字合盘 / 五维评分
- 灵境占卜 — 塔罗 / 摇杯筊 / 每日一签
- 灵境修行 — 了凡打卡 / 冥想 / 能量日记
- 灵境市集 — 数字周边 / 水晶商城 / 线下
- 灵境预言 — 八字排盘 / AI 解盘 / 运势

### 1.2 目标用户
- **年轻白领**（25-40 岁）：对传统命理好奇但需要现代解读方式
- **玄学爱好者**：有八字基础，需要工具辅助快速排盘
- **内容创作者**：需要命理数据作为创作素材

### 1.3 核心价值主张
> "用 AI 读懂你的八字，用 K 线看懂你的人生" — 将抽象的命理概念转化为直观的可视化图表。

### 2.7 灵境人格模块（Module 2 — v3.4 新增）

**核心理念**：基于用户八字日主与十神组合，生成一位"灵魂伴侣式"虚拟人格。AI 以此人格身份与用户进行日常对话、情绪疏导、人生建议。

**人格生成逻辑**：
- **日主映射基础人格**：木→仁者、火→礼者、土→信者、金→义者、水→智者
- **十神赋予特质**：正官（责任）/ 七杀（魄力）/ 正印（慈爱）/ 偏印（神秘）/ 比肩（独立）/ 劫财（豪爽）/ 食神（优雅）/ 伤官（才华）/ 正财（务实）/ 偏财（灵动）
- **性别映射**：根据用户性别生成互补或同性的虚拟伴侣
- **AI Prompt 注入**：将八字特征 + 人格标签注入系统 prompt

**对话能力**：
- 情绪支持：识别用户情绪，给予鼓励或开解
- 命理解读：基于八字给出轻量化解读
- 日常陪伴：早晚安、节气提醒、纪念日
- 玄学问答：可解释梦境 / 塔罗 / 签文含义

**技术要求**：
- 对话历史 SQLite 存储（保留最近 50 轮）
- LLM 调用 + 算法回退（无 Key 时返回预设话术）
- 一键"重置人格"功能

### 2.8 灵境占卜模块（Module 5 — v3.5 新增）

**三种占卜方式**：

#### 2.8.1 塔罗牌阵
- **牌库**：78 张标准韦特塔罗（大阿尔克那 22 + 小阿尔克那 56）
- **牌阵**：单牌（一日指引）、三牌（过去/现在/未来）、凯尔特十字（深度占卜）
- **算法**：随机抽取 + 逆位概率 30%
- **AI 解读**：牌面含义 + 用户问题关联解读

#### 2.8.2 摇杯筊
- **原理**：模拟三圣杯问卜，两枚杯筊投掷
- **结果**：圣杯（一阴一阳=吉笑杯）、阴杯（双阴=凶）、阳杯（双阳=怒）
- **AI 解读**：根据问题给出 1-3 个问题的连续问卜

#### 2.8.3 每日一签
- **签库**：100 支古典签文（基于《观音灵签》《吕祖灵签》改编）
- **内容**：签号、签等（上上/上/中上/中/中下/下/下下）、签诗、典故、解曰
- **算法**：基于日期哈希 + 用户八字生成稳定签文（每日固定）
- **分享**：可生成图片卡片分享

### 2.9 灵境修行模块（Module 6 — v3.6 新增）

**三件套设计**：

#### 2.9.1 了凡四训打卡
- **任务类型**：日行一善 / 诵读经典 / 静坐冥想 / 布施捐赠
- **打卡机制**：每日 3 任务，完成 +1 修行值
- **修行等级**：初学者(0-7) → 行者(8-30) → 居士(31-100) → 真人(101-365) → 仙人(366+)
- **连续打卡**：streak 计数，激励维持习惯

#### 2.9.2 冥想引导
- **场景库**：森林/海边/星空/古寺/雪山 5 大场景
- **引导时长**：3/5/10/20 分钟四档
- **形式**：文字引导 + 呼吸引奏
- **背景音**：可选（Demo 模式无音频但有可视化动画）

#### 2.9.3 能量日记
- **每日记录**：情绪(1-10) / 精力(1-10) / 感恩三件事 / 今日反思
- **趋势图**：7/30/90 天情绪曲线
- **AI 周报**：每周日自动生成能量周报（基于本周日记）

### 2.10 灵境市集模块（Module 7 — v3.7 新增）

**三类商品**：

#### 2.10.1 数字周边
- AI 解梦报告（PDF）

- 塔罗牌阵（PNG 长图）

#### 2.10.2 水晶商城
- **商品库**：紫水晶 / 黄水晶 / 粉晶 / 黑曜石 / 翡翠 / 和田玉等 20+ SKU
- **属性映射**：水晶 → 五行 → 适配日主（如紫水晶=火，适配水命人）

#### 2.10.3 线下活动

- 信息：时间 / 地点 / 名额 / 简介
- 报名：Demo 模式记录意向，不实际收费

### 2.6 缘分配对模块（Module 4 — v3.1 新增）

**合盘算法**：基于双方八字排盘，自动分析五行生克、十神配合、日柱地支关系。

**五维兼容评分**：

| 维度 | 内容 | 计算依据 |
|------|------|----------|
| 性格互补 (personality) | 双方五行相生关系 + 十神加分 | 日干五行生克 60% + 十神 20% |
| 事业合作 (career) | 十神搭配是否理想 | 正官/七杀配正印、偏财等理想搭配 |
| 感情契合 (romance) | 日柱地支六合/六冲关系 | 六合 90 分、六冲 30 分、普通 50 分 |
| 沟通默契 (communication) | 五行流通性 | 相生 vs 相克的沟通对称性 |
| 长期发展 (long_term) | 四维度平均加权 | 综合关系可持续性 |

**关系画像**：
- 🔮 天作之合（>80 分）：日柱六合，前世渊源深厚
- 💫 五行相生（>65 分）：能量流通顺畅，一方滋养另一方
- 🤝 十神相合（>60 分）：事业型最佳拍档
- ⚡ 欢喜冤家（<40 分）：日柱六冲，需空间和耐心
- 🌸 平常之缘（40-60 分）：中正平和，平淡见真情

**五行对比雷达图**：双人五行分布叠加对比，直观展示互补与冲突区域。

**内建降级**：服务器不可用时自动启用 Demo 模式，返回模拟合盘数据。

## 3. 用户流程

```
灵境图谱  → 输入出生信息 → 八字排盘 → 五行分析 → 人生K线
缘分配对  → 输入双方信息 → 五行/十神/地支分析 → 五维评分雷达
梦境回廊  → 录入梦境 → AI 双维解析 → 趋势可视化

灵境占卜  → 选占卜方式 → 牌/签/筊 → AI 解读
灵境修行  → 打卡/冥想/日记 → 修行值累积 → 等级提升
灵境市集  → 浏览商品/活动 → 心愿单 → Demo 模式不实际支付
灵境预言  → 输入八字 → LLM 解盘 → 运势预测
```
