# Nexus Ora — 产品需求文档 (PRD)

> 版本：v3.0 | 更新日期：2026-06-12 | 状态：MVP 已交付

---

## 1. 产品概述

### 1.1 产品定位
Nexus Ora 是一款 **AI 原生玄学平台**，将东方传统命理学（八字排盘、五行分析）与现代 AI 技术结合，为用户提供可视化、数据化的"人生 K 线"运势分析。

### 1.2 目标用户
- **年轻白领**（25-40 岁）：对传统命理好奇但需要现代解读方式
- **玄学爱好者**：有八字基础，需要工具辅助快速排盘
- **内容创作者**：需要命理数据作为创作素材

### 1.3 核心价值主张
> "用 AI 读懂你的八字，用 K 线看懂你的人生" — 将抽象的命理概念转化为直观的可视化图表。

## 2. 功能需求

### 2.1 MVP 已实现功能

| 模块 | 功能 | 优先级 | 状态 |
|------|------|--------|------|
| 八字排盘 | 输入出生日期/时间/性别，自动生成四柱八字 | P0 | ✅ |
| 五行分析 | 天干地支五行属性识别，五行平衡统计 | P0 | ✅ |
| 十神识别 | 以日干为基准计算十神关系 | P0 | ✅ |
| 身强身弱 | 基于五行分布自动判定日主强弱 | P0 | ✅ |
| 人生K线 | 101 点（0-100 岁）运势曲线可视化 | P0 | ✅ |
| AI 解读 | 六维度深度解读（事业/财运/感情/健康/贵人/挑战） | P1 | ✅ |
| 算法回退 | 无 API Key 时自动启用五行算法模式 | P0 | ✅ |
| 付费解锁 | Stripe 支付，Demo 模式一键体验 | P1 | ✅ |
| 数据持久化 | 用户数据本地 SQLite 存储 | P1 | ✅ |
| 离线预览 | 纯前端 HTML 可直接打开体验 | P1 | ✅ |

### 2.2 运势曲线颜色规则
遵循**中国股市惯例**（与欧美色系相反）：
- 🔴 **红色** = 运势上涨（大吉/小吉）
- 🟢 **绿色** = 运势下跌（大凶/小凶）
- ⚪ **灰色** = 平稳

### 2.3 AI 解读六个维度

| 维度 | 内容 | 评分范围 |
|------|------|----------|
| 事业 (career) | 职业方向建议、行业适配 | 1-95 |
| 财运 (wealth) | 理财风格、财富周期 | 1-95 |
| 感情 (relationships) | 感情特质、最佳婚配 | 1-95 |
| 健康 (health) | 身体养护重点、季节调养 | 1-95 |
| 贵人 (mentors) | 贵人运势、社交建议 | 1-92 |
| 挑战 (challenges) | 人生低谷期、应对策略 | 1-90 |

### 2.4 运势等级定义

| 等级 | 分数范围 | 含义 |
|------|----------|------|
| 大吉 | ≥ 75 | 运势旺盛，适宜行动 |
| 小吉 | 60-74 | 运势良好，稳步前进 |
| 平稳 | 45-59 | 运势平淡，以守为主 |
| 小凶 | 30-44 | 运势低迷，谨慎行事 |
| 大凶 | < 30 | 运势凶险，建议回避 |

### 2.5 人生阶段划分

| 阶段 | 年龄段 | 特征 |
|------|--------|------|
| 成长期 | 0-18 | 基础教育，性格塑造 |
| 上升期 | 19-30 | 学业完成，职场起步 |
| 黄金期 | 31-45 | 事业巅峰，财富积累 |
| 稳固期 | 46-60 | 事业稳定，家庭成熟 |
| 智慧期 | 61-75 | 经验传承，享受成果 |
| 颐养期 | 76-100 | 安享晚年，精神富足 |

## 3. 用户流程

```
用户输入出生信息 → 八字排盘 → 五行分析 → [AI 解读 / 算法回退]
                                  ↓
                          人生K线可视化
                                  ↓
                        部分内容预览（免费）
                                  ↓
                        付费解锁完整报告
```
