## 1. 产品概述

### 1.1 产品定位
Nexus Ora 是一款 **AI 原生玄学平台**，将东方传统命理学（八字排盘、五行分析）与现代 AI 技术结合，构建"灵境八部"产品矩阵：
- 灵境图谱 — 人生K线 / 五行平衡 / 八字排盘
- 灵境人格 — 十神虚拟伴侣 AI 对话
- 梦境回廊 — AI 梦境记录 / 可视化 / 解析
- 缘分配对 — 双人八字合盘 / 五维评分
- 灵境占卜 — 塔罗 / 摇杯筊 / 每日一签
- 灵境修行 — 了凡打卡 / 冥想 / 能量日记
- 灵境市集 — 数字周边 / 水晶商城 / 线下
- 灵境预言 — 八字排盘 / AI 解盘 / 运势

### 1.2 目标用户
- **年轻白领**（25-40 岁）：对传统命理好奇但需要现代解读方式
- **玄学爱好者**：有八字基础，需要工具辅助快速排盘
- **内容创作者**：需要命理数据作为创作素材

### 1.3 核心价值主张
> "用 AI 读懂你的八字，用 K 线看懂你的人生" — 将抽象的命理概念转化为直观的可视化图表。

### 2.7 灵境人格模块（Module 2 — v3.4 新增）

**核心理念**：基于用户八字日主与十神组合，生成一位"灵魂伴侣式"虚拟人格。AI 以此人格身份与用户进行日常对话、情绪疏导、人生建议。

**人格生成逻辑**：
- **日主映射基础人格**：木→仁者、火→礼者、土→信者、金→义者、水→智者
- **十神赋予特质**：正官（责任）/ 七杀（魄力）/ 正印（慈爱）/ 偏印（神秘）/ 比肩（独立）/ 劫财（豪爽）/ 食神（优雅）/ 伤官（才华）/ 正财（务实）/ 偏财（灵动）
- **性别映射**：根据用户性别生成互补或同性的虚拟伴侣
- **AI Prompt 注入**：将八字特征 + 人格标签注入系统 prompt

**对话能力**：
- 情绪支持：识别用户情绪，给予鼓励或开解
- 命理解读：基于八字给出轻量化解读
- 日常陪伴：早晚安、节气提醒、纪念日
- 玄学问答：可解释梦境 / 塔罗 / 签文含义

**技术要求**：
- 对话历史 SQLite 存储（保留最近 50 轮）
- LLM 调用 + 算法回退（无 Key 时返回预设话术）
- 一键"重置人格"功能

### 2.8 灵境占卜模块（Module 5 — v3.5 新增）

**三种占卜方式**：

#### 2.8.1 塔罗牌阵
- **牌库**：78 张标准韦特塔罗（大阿尔克那 22 + 小阿尔克那 56）
- **牌阵**：单牌（一日指引）、三牌（过去/现在/未来）、凯尔特十字（深度占卜）
- **算法**：随机抽取 + 逆位概率 30%
- **AI 解读**：牌面含义 + 用户问题关联解读

#### 2.8.2 摇杯筊
- **原理**：模拟三圣杯问卜，两枚杯筊投掷
- **结果**：圣杯（一阴一阳=吉笑杯）、阴杯（双阴=凶）、阳杯（双阳=怒）
- **AI 解读**：根据问题给出 1-3 个问题的连续问卜

#### 2.8.3 每日一签
- **签库**：100 支古典签文（基于《观音灵签》《吕祖灵签》改编）
- **内容**：签号、签等（上上/上/中上/中/中下/下/下下）、签诗、典故、解曰
- **算法**：基于日期哈希 + 用户八字生成稳定签文（每日固定）
- **分享**：可生成图片卡片分享

### 2.14 钦天四化模块（北派飞星 — v5.1 新增）

**背景**：紫微斗数传统分"三合派"与"四化派（飞星派/钦天派）"。现有紫微斗数引擎（v3.2）以"星曜格局 + 生年四化"为主，偏三合视角。钦天四化是紫微斗数中体系最精密、逻辑最自洽的飞星流派，以**四化（禄权科忌）为第一性**，宫位与星曜为承载，强调"来因定根基、生年定天赋、自化定开关"。

**核心概念**：

| 概念 | 含义 | 产品呈现 |
|------|------|----------|
| 来因宫 | 宫干 == 年干之宫，代表"我为何而来"的生命主题 | 紫色高亮卡片，标注宫名+宫干 |
| 生年四化 | 年干决定的 化禄(A)/化权(B)/化科(C)/化忌(D) 四星及其落宫 | A/B/C/D 四宫卡，显示"星 → 落宫" |
| 宫干飞星 | 十二宫各有宫干，每宫四化飞入他宫，共 48 条飞化轨迹 | 可切换宫位查看该宫四化飞往何处 |
| 离心自化 | 本宫宫干所化之星恰落本宫，能量"出而不留" | 标签展示 宫·ABCD·星 |
| 向心自化 | 对宫宫干所化之星飞入本宫，能量"自外挤压" | 标签展示 宫·ABCD·自X宫 |

**四化象义（钦天核心：无吉凶，只分态度）**：
- A 化禄：缘起、多缘、喜悦（生发）
- B 化权：变化、争权、善变（掌控）
- C 化科：情份、念旧、解厄（贵气）
- D 化忌：亏欠、执迷、阻滞（课题）

**用户价值**：
- 与现有"三合格局"互补：三合看"星曜组合定性格"，钦天看"四化流动定事件契机"，两套视角交叉验证
- 比单一"生年四化"更深：引入宫干飞星与自化，能解释"为何同星不同命""为何某些宫位能量留不住"
- 差异化卖点：市面多数排盘工具只给出生年四化，钦天四化（来因+飞星+自化）是进阶刚需

**技术实现**：纯算法（无 LLM 依赖），复用现有 `ziwei_engine.js` 的排盘结果与四化表，新增 `computeQinTian()`：`/api/ziwei` 返回 `qinTian` 对象（来因宫/生年四化/48 条飞星/离心自化/向心自化）。前端 Web 与小程序均新增"钦天四化"展示区。详见 TRD §3.7。

**验收标准**：
- 来因宫计算正确（五虎遁安宫干，宫干==年干唯一定位）
- 生年四化落宫与四化表一致
- 48 条飞星轨迹无遗漏、无错位
- 离心/向心自化识别准确
- Web + 小程序均可见、可交互

### 2.6 缘分配对模块（Module 4 — v3.1 新增）

**合盘算法**：基于双方八字排盘，自动分析五行生克、十神配合、日柱地支关系。

**五维兼容评分**：

| 维度 | 内容 | 计算依据 |
|------|------|----------|
| 性格互补 (personality) | 双方五行相生关系 + 十神加分 | 日干五行生克 60% + 十神 20% |
| 事业合作 (career) | 十神搭配是否理想 | 正官/七杀配正印、偏财等理想搭配 |
| 感情契合 (romance) | 日柱地支六合/六冲关系 | 六合 90 分、六冲 30 分、普通 50 分 |
| 沟通默契 (communication) | 五行流通性 | 相生 vs 相克的沟通对称性 |
| 长期发展 (long_term) | 四维度平均加权 | 综合关系可持续性 |

**关系画像**：
- 🔮 天作之合（>80 分）：日柱六合，前世渊源深厚
- 💫 五行相生（>65 分）：能量流通顺畅，一方滋养另一方
- 🤝 十神相合（>60 分）：事业型最佳拍档
- ⚡ 欢喜冤家（<40 分）：日柱六冲，需空间和耐心
- 🌸 平常之缘（40-60 分）：中正平和，平淡见真情

**五行对比雷达图**：双人五行分布叠加对比，直观展示互补与冲突区域。

**内建降级**：服务器不可用时自动启用 Demo 模式，返回模拟合盘数据。

## 4. 商业模式

| 方案 | 价格 | 内容 |
|------|------|------|
| 免费会员 (free) | $0 | 八字排盘 + 5 年运势预览，基础功能 |
| 单次报告 (report_only) | $9.99 | 解锁当前深度报告，永久有效 |
| 月度会员 (premium_monthly) | $9.99/月 | 全模块无限次排盘 + AI 深度解读 + 完整报告 |
| 年度会员 (premium_annual) | $59.99/年 | 月度全部权益 + 优先支持（较月度省 ~50%） |

### 支付渠道
- **Stripe**：信用卡支付（Web 端）
- **PayPal**：全球支付（Web 端 + 订阅，按钮黄色 #FFC439）
- 会员等级存储于 `auth_users` 表（`membership_tier` / `expires` / `started`），`checkMembership()` 做过期检查
- 付费接口用会员等级做权限校验，所有 SQL 查询参数化（防注入）

### Demo 模式
未配置任何支付 / LLM API Key 时自动启用，所有功能免费可用（含"模拟支付"一键解锁）。
